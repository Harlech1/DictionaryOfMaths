matrix polynomial

A polynomial with matrix coefficients.

An nnth order matrix polynomial in a variable xx is given by P(x)=Ao+A1x+A1x2+…+AnxnP(x) = A_{o} + A_{1}x + A_{1}x^{2} + \ldots + A_{n}x^{n} where Ao,A1,A1,…,AnA_{o},{\ A}_{1},\ {\ A}_{1},\ \ldots,\ A_{n} are square matrices of the same dimensions.
