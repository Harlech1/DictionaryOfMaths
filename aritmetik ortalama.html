arithmetic mean; average

The average of a set of measures is the quotient of the sum of the measures in the set and the number of elements of the set. If there are nn observations x1,x2,..,xnx_{1},x_{2},..,x_{n}, then arithmeticmean=x1+x2+..,+xnn=∑i=1nxinarithmeticmean = \frac{x_{1} + x_{2} + ..,{+ x}_{n}}{n} = \frac{\sum_{i = 1}^{n}x_{i}}{n}
