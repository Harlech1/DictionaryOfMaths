2.base (of a logarithm)

The number system in which a logarithm is computed.

The logarithm of a number x in base b is written as logbx\log_{b}x, where x>0,b>0x > 0,\ b > 0 and b≠1b \neq 1.

3. base (of a number system)

A real number x can be represented using any integer number b as a base. In base b, the digits 0, 1, 2, 3, . . . , b-1 are used.
