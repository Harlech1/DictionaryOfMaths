standard deviation

The standard deviation is a measure of the spread of scores within a set of data.

N: number of scores in the data xix_{i}: each score x¯\overline{x}: mean of the scores

Population standard deviation: s=∑i=1N(xi−x¯)2Ns = \sqrt{\frac{\sum_{i = 1}^{N}{(x_{i} - \overline{x})}^{2}}{N}} Sample standard deviation: s=∑i=1N(xi−x¯)2N−1s = \sqrt{\frac{\sum_{i = 1}^{N}{(x_{i} - \overline{x})}^{2}}{N - 1}}
