properties of the exponential function f(x)=axf(x) = a^{x}, a>1or0<a<1a > 1\ \ or\ 0 < a < 1\

Its domain is the set of all real numbers. Its range is the set of positive real numbers.

There are no x-intercepts and y-intercept is 1. x-axis is the horizontal asymptote.

If a >1, then it is an increasing function and is 1-1. If 0< a <1, then it is a decreasing function and is 1-1.

Its graph is continuous, with no corners or gaps.
