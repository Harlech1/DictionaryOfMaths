partial derivative

Derivative of a function of multiple variables when all but the variable of interest are held fixed during differentiation.

When a function f(x,y,z,...)f(x,y,\ z,\ \ .\ \ .\ \ .\ ) depends on more than one variable, a partial derivative ∂f∂x\frac{\partial f}{\partial x}\ , ∂f∂y\frac{\partial f}{\partial y} , ∂2f∂x∂y\frac{\partial^{2}f}{\partial x\ \partial y} , etc. can be used. fx=∂f∂x=limh→0f(x+h,y,z,...)−f(x,y,z,...)hf_{x} = \frac{\partial f}{\partial x} = \lim_{h \rightarrow 0}\frac{f(x + h,y,z,\ .\ .\ .\ ) - f(x,y,z,\ .\ .\ .)}{h} , fy=∂f∂y=limh→0f(x,y+h,z,...)−f(x,y,z,...)h\ f_{y} = \frac{\partial f}{\partial y} = \lim_{h \rightarrow 0}\frac{f(x,y + h\ ,\ z,\ .\ .\ .\ ) - f(x,y,z,\ .\ .\ .)}{h} and so on.
