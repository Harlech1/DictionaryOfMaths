probability

The chance that a particular event (or set of events) will occur expressed as a linear scale from 0 (impossibility) to

1 (certainty). It can be expressed as a fraction, decimal, or percent. The probability of an event is the ratio of the number of outcomes favoring the event to the total number of possible outcomes.

If all of the outcomes in a sample space SS are equally likely, then the probability of an event AA is defined by

P(A)=n(A)n(S)P(A) = \frac{n(A)}{n(S)} , where n(S)\ n(S) is the total number of outcomes in SS and n(A)n(A) is the number of outcomes favoring the event A. The sum of the probabilities of all possible outcomes is always 1.
