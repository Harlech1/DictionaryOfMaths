prob. variance

It measures how far a set of numbers is spread out. The variance of a set of nn equally likely values is given by

Var(X)=1n∑i=1n(xi−x¯)2Var(X) = \frac{1}{n}\sum_{i = 1}^{n}{(x_{i} - \overline{x})}^{2}, where x¯\overline{x} is the mean, i.e. x¯=1n∑i=1nxi\overline{x} = \frac{1}{n}\sum_{i = 1}^{n}\ x_{i}.
