transforming the function

I. To replace the old function f(x)f(x) by f(x+a)f(x + a), for some number aa (horizontal shift by aa).

II. To add some number aa outside the function, taking f(x)f(x) to f(x)+af(x) + a (vertical shift by aa).

III. To multiply the function f(x)f(x) by a constant aa, to get af(x)af(x) (vertical stretch by aa).

IV. To go from f(x)f(x) to f(ax)f(ax) (horizontal shrink by aa)
