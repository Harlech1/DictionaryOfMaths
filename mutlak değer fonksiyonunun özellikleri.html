properties of the absolute value function f(x)=|x|f(x) = |x|\

The domain is the set of all real numbers. The range is the set of nonnegative real numbers.

It is an even function. The graph is symmetric with respect to the y-axis.

It is decreasing on the interval (−∞,0)( - \infty,0) and increasing on the open interval (0,∞)(0,\infty). It has a local minimum of 0 at x=0.
