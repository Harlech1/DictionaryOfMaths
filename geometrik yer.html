locus pl.\ pl.\ loci

The set of points which satisfies a set of criteria is called the locus of the criteria. The locus of

(a) all points in a plane equidistant from a given point is a circle;

(b) all points in 3-space equidistant from a given point is a sphere;

(c) all points in a plane equidistant from two intersecting lines is the pair of angle bisectors formed by these two lines;

(d) the center of a variable circle, tangent to a given circle and passing through a given point inside that circle, is an ellipse;

(e) the center of a variable circle, tangent to a given circle and passing through a given point outside that circle, is a hyperbola;

(f) the center of a variable circle, tangent to a given line and passing through a given point not on that line, is a parabola.
